## 拆分指南

Spec 模板是单文件示意，展示完整结构。实际编写时，按以下规则拆分：

- 如果某节的内容可独立阅读和理解（如接口按模块分组）→ 独立为文件
- 如果某节内部有二级结构（如数据模型有多个实体）→ 独立为文件
- 拆分后，本节改为引用链接，不复制内容
- 不拆分的情况：内容简单、无内部结构、与上下文紧密耦合
